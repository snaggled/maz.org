I've been building web applications and services for 15 years. Some of my most interesting and challenging projects include: one of the web's earliest community sites, a web-based personal information manager, an advertising network for PC and console games, an open source calendar server, and the public API for a hyperlocal content aggregation service. My focus is on finding better ways for web-based software components to interact with each other to enable people to do interesting and useful things, and I find passion in building such systems with elegance and style.
